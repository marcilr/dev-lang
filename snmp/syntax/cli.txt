cli -   Command line interface for snmp


o snmptranslate: learning about the MIB tree.
o snmpget: retrieving data from a host.
o snmpgetnext: retrieving unknown indexed data.
o snmpwalk: retrieving lots of data at once!
o snmptable: displaying a table.
o snmpset: peforming write operations.
o snmpbulkget: communicates with a network entity using SNMP GETBULK request
o snmpbulkwalk: retrieve a sub-tree of management values using SNMP GETBULK
  requests.
o snmptrap: Sending and receiving traps, and acting upon them.
